PHP

PHP adalah bahasa scripting server-side yang digunakan dalam pembuatan website bersama dengan CSS dan HTML. Bahasa pemrograman ini digunakan untuk mengembangkan situs web statis, situs web dinamis atau aplikasi Web. PHP singkatan dari Hypertext Pre-processor, yang sebelumnya disebut Personal Home Pages. PHP berjalan di berbagai platform (Windows, Linux, Unix, Mac OS X).

LARAVEL

Laravel adalah sebuah framework PHP yang dirilis dibawah lisensi MIT, dibangun dengan konsep MVC (model view controller). Laravel adalah pengembangan website berbasis MVP yang ditulis dalam PHP yang dirancang untuk meningkatkan kualitas perangkat lunak dengan mengurangi biaya pengembangan awal dan biaya pemeliharaan, dan menyediakan sintaks yang ekspresif, jelas dan menghemat waktu.

MySQL

MySQL adalah sebuah database management system (manajemen basis data) dengan menggunakan perintah dasar SQL (Structured Query Language) yang cukup terkenal. Database management system (DBMS) MySQL multi pengguna dan multi alur ini sudah dipakai lebih dari 6 juta pengguna di seluruh dunia. MySQL tersedia sebagai perangkat lunak gratis di bawah lisensi GNU General Public License (GPL).

JAVA SCRIPT

halaman web menggunakan tag SCRIPT. JavaScript merupakan teknologi inti World Wide Web selain HTML dan CSS. JavaScript membantu membuat halaman web interaktif dan merupakan bagian aplikasi web yang esensial. Javascript berfungsi untuk berinteraksi antara situs web dengan user menjadi lebih cepat.

ANDROID

komputer tablet yang dirilis oleh Google, dengan kode di bawah Lisensi Apache. Android bersifat open source dapat dimodifikasi, diperbaiki dan didistribusikan oleh para pembuat ataupun pengembang perangkat lunak. Dengan open source perusahaan teknologi bebas menggunakan OS diperangkatnya tanpa harus lisensi.

Apple iOS

atau GUI. iOS hanya digunakan pada perangkat Apple, seperti iPhone, iPad dan iPod Touch. Swift merupakan bahasa pemrograman objek fungsional untuk pengembangan iOS dan OS X yang dibuat oleh Apple. Didesain untuk berdampingan dengan Objective-C dan menghindari program dari kode yang salah.

BOOTSTRAP

Bootstrap adalah library (pustaka/kumpulan fungsi-fungsi) dari Framework CSS yang dibuat khusus untuk bagian pengembangan front-end dari suatu website. Kerangka kerja yang berisi template desain berbasis HTML dan CSS untuk tipografi, formulir, tombol, navigasi, dan komponen antarmuka lainnya, serta juga ekstensi opsional JavaScript. Bootstrap dapat memudahkan dan mempercepat pembuatan website.

APACHE CORDOVA

Apache Cordova adalah perangkat lunak untuk membangun web mobile applications menggunakan bahasa pemrograman CSS3, HTML5, dan JavaScript, dengan mengandalkan API khusus platform seperti yang ada di Android, iOS, atau Windows Phone. Dengan satu kode sumber yang sama, Apache Cordova dapat membungkus aplikasi agar dapat berjalan di banyak jenis device.

Node.js

Node.js adalah perangkat lunak yang didesain untuk mengembangkan suatu aplikasi yang berbasis web dan ditulis dalam sintaks bahasa pemrograman JavaScript. Node.js ini salah satu platform yang memiliki kualitas yang cukup tinggi. Node.js dibangun dengan pondasi bahasa pemograman javascript. Node.js dapat berjalan di sistem operasi Windows, Mac OS X dan Linux tanpa perlu ada perubahan kode program.
